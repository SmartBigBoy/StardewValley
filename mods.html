Prepare before installing mods

📁

Installation Location

Steam Version:

C:\Program Files (x86)\Steam\steamapps\common\Stardew Valley\Mods

GOG Version:

C:\GOG Games\Stardew Valley\Mods
⚙️

Required Dependencies

  • SMAPI - Mod loader, required
  • Content Patcher - Framework mod
  • SpaceCore - Compatibility Framework

Download:smapi.io

⚠️ Resolving Mod Conflicts

Solve common mod conflict issues

Common Conflict Causes

  • Multiple mods editing the same file
  • SMAPI version incompatibility
  • Missing mod dependencies
  • Modifying outdated game files

Solutions

  • Use mod manager to check conflicts
  • Make sure SMAPI is the latest version
  • Check compatibility list in mod description
  • Install only a few mods at a time for testing
