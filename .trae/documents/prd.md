# 双人成行攻略站 PRD文档

## 1. 产品概述

双人成行（It Takes Two）攻略站是一款专为Switch/PC平台《双人成行》游戏玩家打造的专业攻略指南网站。网站提供全章节图文攻略、Boss打法技巧、25个小游戏位置指引及双人配合心得，帮助玩家高效通关并解锁全部成就。

- 目标用户：双人成行游戏玩家，尤其是需要攻略指引的新手玩家
- 核心价值：一站式获取完整、准确、实用的游戏攻略

## 2. 核心功能

### 2.1 功能模块

1. **首页**：展示游戏概览、章节导航、热门攻略推荐
2. **章节攻略**：7大章节详细图文攻略，含关卡流程、解谜技巧
3. **Boss战指南**：11+个Boss完整打法、攻击模式解析
4. **小游戏大全**：25个小游戏位置与玩法详解
5. **配合技巧**：双人合作核心技巧、避坑要点
6. **搜索功能**：支持关键词搜索，快速定位攻略内容

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 首页 | Hero区域 | 展示游戏封面、标题、快速入口 |
| 首页 | 章节导航卡片 | 7大章节快捷入口 |
| 首页 | 热门攻略 | 推荐高点击量攻略文章 |
| 章节攻略页 | 章节概览 | 章节主题、时长、核心机制 |
| 章节攻略页 | 关卡详情 | 每个小节的详细图文流程 |
| Boss页 | Boss列表 | 所有Boss快速导航 |
| Boss页 | Boss详情 | 攻击模式、应对策略、视频演示 |
| 小游戏页 | 位置导航 | 25个小游戏章节分布 |
| 小游戏页 | 游戏详情 | 玩法说明、得分技巧 |
| 配合技巧页 | 核心技巧 | 四大万能技巧 |
| 配合技巧页 | 避坑要点 | 各章节常见问题 |
| 全站 | 搜索栏 | 实时搜索、热门关键词 |

## 3. 核心流程

```
用户访问 → 首页 → 选择章节/Boss/小游戏
     ↓
搜索栏输入 → 实时匹配 → 展示结果 → 点击查看详情
     ↓
攻略详情页 → 阅读图文 → 收藏/分享
```

## 4. 用户界面设计

### 4.1 设计风格

- **主题色**：深紫色(#6B4C9A) + 暖橙色(#FF6B35)，呼应游戏奇幻温馨风格
- **背景色**：深灰渐变(#1a1a2e → #16213e)，营造沉浸感
- **卡片风格**：毛玻璃效果、圆角边框、柔和阴影
- **字体**：Noto Sans SC（中文）+ Poppins（英文标题）
- **布局**：卡片式布局、响应式分栏

### 4.2 视觉元素

- 图标：手柄、爱心、星星等游戏元素图标
- 动效：淡入滑动、悬停放大、点击反馈
- 装饰：星光粒子背景、音波图形装饰

### 4.3 响应式设计

- 桌面端：多栏布局，充分利用屏幕空间
- 平板端：自适应双栏
- 移动端：单栏堆叠，底部导航

## 5. 技术架构

- **前端**：纯HTML5 + CSS3 + Vanilla JavaScript
- **无需后端**：所有内容静态存储
- **无需数据库**：JSON数据存储攻略内容
