🌐 Multiplayer Methods

Introduction to different multiplayer methods

🌐

LAN Multiplayer

Players under the same network can connect directly:

  • Host selects "Invite"
  • Joiner selects "Join"
  • Select the host on the same network
💻

Online Multiplayer

Plant & Harvest or use Steam friend system:

  • Invite friends via Steam
  • Use Hamachi or similar VPN
  • Port forwarding settings

Multiplayer Note Matters

Resource rules in multiplayer mode

Shared Resources

  • Shared gold (independent wallet optional)
  • Players can trade items with each other
  • Shipping box income is shared

Independent Resources

  • Each player has their own inventory
  • Each player has independent tools
  • Independent cabins (additional cabins)
  • Independent friendship progress
